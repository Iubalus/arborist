# <img src="arborist.png" width="30"> Setup Instructions


* Set up a new MongoDB or locate an existing DB
* Copy `src/main/resources/application.property.sample` into `src/main/resources/application.property` in the same directory
* Replace placeholder values with real values
  * Replace `<your mongo db uri>` with an actual URI to the MongoDB
  * Replace `<your mongo db name>` with an actual DB Name
